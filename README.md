# UniServe – Canteen Pre-Order & Crowd Control System

## 📌 Project Overview

**UniServe** is a web-based Canteen Pre-Order and Crowd Control System designed to solve common problems faced by university students and canteen staff such as long waiting queues, overcrowding, delayed order processing, and inefficient feedback management.

The system allows students to pre-order meals, manage their profiles, provide feedback, and receive notifications, while administrators can manage users, monitor feedback, generate reports, and maintain system operations efficiently.

---

# 🎯 Real-World Problems Solved

### Traditional Canteen Problems

* Long queues during peak hours.
* Students wasting time waiting for food.
* Overcrowding inside the canteen.
* Difficulty managing customer feedback.
* Manual order handling.
* Lack of user management and reporting.

### UniServe Solution

* Online meal pre-ordering.
* Reduced waiting time.
* Better crowd management.
* Digital feedback collection.
* User account management.
* Administrative reports and analytics.

# 🚀 Installation Guide

## Clone Repository

```bash

```

---

## Backend Setup

### Navigate

```bash
cd Backend
```

### Install Dependencies

```bash
npm install
```

### Create .env File

```env

JWT_SECRET ="jwt-secret"

EMAIL_USER = "nikidesilva704@gmail.com"
```

### Start Backend

```bash
npm run dev
```

Server runs on:

```text
http://localhost:5000
```

---

## Frontend Setup

### Navigate

```bash
cd Frontend
```

### Install Dependencies

```bash
npm install
```

### Create .env File

```env
VITE_API_URL=http://localhost:5000

VITE_ORDER_PAGE_URL=http://localhost:5000/order
```

### Start Frontend

```bash
npm run dev
```

Application runs on:

```text
http://localhost:5173
```
